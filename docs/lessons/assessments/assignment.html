Level 1 Assignment Review

Assignment Review

Students are to schedule a level assignment review after completion of the Escape Room Assignment in level 1.

Assignments:

  • DOM Olympics
  • Mom's Shopping List
  • Escape Room

Requirements:

  • All three assignments must be completed and ready to show to the best of their ability.
  • Students should be prepared to talk about the concepts and functionality used in each assignment.
  • Level Assignment Review Submission is required before scheduling or passing a Final Assessment.

Review Process:

  • The review with notes and feedback of your work will be sent over to you via discord.
  • During the Level Assessment the instructor will ask questions about the code and the student's understanding of the concepts from these assignments.

Benefits of the Review:

  • The review is an important part of the learning process.
  • It's a chance for students to get feedback on their work and to improve their skills.
  • The review also helps students to prepare for grading of future assignments and assessments.

Questions?

Please contact the instructor if you have any questions about the Level Assignment Review.

Additional Notes:

  • The instructor may ask students to share their screen during the review.
  • The call will be recorded as part of the learning process.

Submitting Assignments

Please remember to regularly submit your assignments to GitHub!

Using Git Command Line:

  • Open Terminal and navigate to your project folder.
  • Add Changes: git add <relative/path/to/assignment/folder>
  • Commit Changes: git commit -m "submitting assignment <name>"
  • Push to GitHub: git push origin main

git add <relative/path/to/assignment/folder>
git commit -m "submitting assignment <name>"
git push origin main

Using VSCode's Source Control:

  • Open VSCode and click on the Source Control tab.
  • Stage Changes: Click the "+" icon next to each file.
  • Commit Changes: Enter a commit message and click Commit.
  • Push to GitHub: Click the Sync Changes button.
