Elements, Console, and Sources Tab

Lesson

Learning Objectives

Students will:

  • Be able to explain the purpose of each developer tab: Console, Elements, and Sources.
  • Be able to use the Console for viewing program output and writing code snippets.
  • Be able to find specific elements in the DOM and read their properties from the Elements tab.
  • Be able to execute browser code using the debugger in Sources tab.
  • Be able to implement breakpoints and use the debugger controls to debug their code

The dev tools are extremely important for problem solving. Watch at 2X speed to learn some tricks for helping you solve the challenges in the DOM exercises!
