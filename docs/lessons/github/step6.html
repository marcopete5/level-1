Adding Changes with git add

What is git add?

  • Explanation: git add is like telling Git, "Hey, I want to include these changes in my next snapshot."
  • Example: Imagine you're preparing a presentation. git add is like selecting the slides you want to include before saving your presentation.
  • Why it matters: It allows you to choose which changes you want to save and which ones you want to leave out for now.

How to Use git add

  1. Open Terminal or Command Prompt:
    • Navigate to your project folder using the cd command.
  2. Type git add <file>:
    • Replace <file> with the name of the file you want to include in the next commit. If you want to add all files, you can use git add ..
  3. Check Status:
    • After adding files, you can use git status to see which files have been staged for the next commit.

Why Use git add?

  • Selective Staging:
    • git add allows you to selectively choose which changes you want to include in the next commit.
  • Control Over Changes:
    • It gives you control over what gets saved, helping you avoid including unfinished or unwanted changes.
  • Incremental Commits:
    • By adding changes in stages, you can make smaller, incremental commits, which can make it easier to track the history of your project.
