Git Status

Lesson

What is git status?

  • Explanation: git status is like asking Git, "What's going on?"
  • Example: Imagine you're in a messy room and you want to know what's out of place. git status is like turning on the lights and seeing what needs to be cleaned up.
  • Why it matters: It helps you keep track of changes you've made to your project and what still needs to be done.

How to Use git status

  1. Open Terminal or Command Prompt:
    • Open the program where you type commands to your computer (like Command Prompt on Windows or Terminal on macOS/Linux).
  2. Navigate to Your Project:
    • Use the cd command to move to the folder where your Git project is located.
  3. Type git status:
    • Once you're in the right folder, just type git status and press Enter.
  4. Understanding the Output:
    • Git will show you a list of files that have been modified, added, or deleted since the last commit.
    • It will also tell you which branch you're on and if there's anything you need to do, like commit changes or add new files.

Why Use git status?

  • Stay Organized:
    • git status helps you stay organized by showing you what's changed since the last time you saved your work.
  • Avoid Surprises:
    • It prevents surprises by letting you know if there's anything you've forgotten to do before sharing your changes with others.
  • Track Progress:
    • You can use it to track your progress and see how far along you are in completing a task or fixing a bug.
